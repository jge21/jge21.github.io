ES6新增语法

发表于2018-08-05|更新于2021-05-22|JavaScript
|阅读量:

1、let、const、var 的区别

  1. 使用 var 声明的变量，其作用域为该语句所在的函数内，且存在变量提升现象
  2. 使用 let 声明的变量，其作用域为该语句所在的代码块内，不存在变量提升
  3. 使用 const 声明的是常量，在后面出现的代码中不能再修改该常量的值
var let const
函数级作用域 块级作用域 块级作用域
变量提升 不存在变量提升 不存在变量提升
值可更改 值可更改 不值可更改

2、解构赋值

  • 他是一种针对数组或者对象进行模式匹配，然后对其中的变量进行赋值。
  • 解构赋值就是把数据结构分解，然后给变量进行赋值
  • 如果结构不成功，变量跟数值个数不匹配的时候，变量的值为undefined
  • 数组解构用中括号包裹，多个变量用逗号隔开，对象解构用花括号包裹，多个变量用逗号隔开

3、箭头函数

  • 函数体中只有一句代码，且代码的执行结果就是返回值，可以省略大括号
  • 如果形参只有一个，可以省略小括号
  • 箭头函数不绑定this关键字，箭头函数中的this，指向的是函数定义位置的上下文this
  • 可以简单理解成，定义箭头函数中的作用域的this指向谁，它就指向谁 (对象没有作用域)

4、剩余参数

  • 剩余参数语法允许我们将一个不定数量的参数表示为一个数组，

  • 不定参数定义方式，这种方式很方便的去声明不知道参数情况下的一个函数

6、扩展运算符（展开语法）

1、扩展运算符可以将数组或者对象转为用逗号分隔的参数序列

2、扩展运算符可以应用于合并数组

2、将类数组或可遍历对象转换为真正的数组

7、构造函数方法：Array.from()

  • 将伪数组或可遍历对象转换为真正的数组
  • 方法还可以接受第二个参数，作用类似于数组的map方法，用来对每个元素进行处理，将处理后的值放入返回的数组

8、实例方法：find()

  • 用于找出第一个符合条件的数组成员，如果没有找到返回undefined

9、实例方法：findIndex()

  • 用于找出第一个符合条件的数组成员的位置，如果没有找到返回**-1**

10、实例方法：includes()

  • 判断某个数组是否包含给定的值，返回布尔值。

11、实例方法：repeat()

  • repeat方法表示将原字符串重复n次，返回一个新字符串

12、Set 数据结构

  • 它类似于数组，但是成员的值都是唯一的，没有重复的值。

  • Set本身是一个构造函数，用来生成 Set 数据结构

  • Set函数可以接受一个数组作为参数，用来初始化。

1、实例方法

  • add(value)：添加某个值，返回 Set 结构本身
  • delete(value)：删除某个值，返回一个布尔值，表示删除是否成功
  • has(value)：返回一个布尔值，表示该值是否为 Set 的成员
  • clear()：清除所有成员，没有返回值

2、遍历

Set 结构的实例与数组一样，也拥有forEach方法，用于对每个成员执行某种操作，没有返回值。

13、模板字符串

  • 使用反引号定义
  • 模板字符串中可以解析变量
  • 模板字符串中可以换行
  • 在模板字符串中可以调用函数·

14、实例方法：startsWith() 和 endsWith()

  • startsWith()：表示参数字符串是否在原字符串的头部，返回布尔值
  • endsWith()：表示参数字符串是否在原字符串的尾部，返回布尔值

后期添加的

数组操作

push：将元素添加到数组的末尾。返回值数组的长度

pop：将数组的最后一个元素弹出，，返回值被弹出的元素

unshift:在数组的开头插入一个元素。返回值数组的长度

shift：将数组的第一个元素弹出，返回值被弹出的元素

splice（index，length）：删除数组中指定的元素，返回删除后的项目，该方法会改变原始数组

conact：连接数组

reverse：反转数组

字符操作：

substr（start，length）截取字符串 会改变原数组 返回删除后的字符

indexOf:查找某一个字符是否存在另一个字符串中，存在返回索引，不存在返回-1 （indexOf从前向后查找，lastIndexOf从后向前查找）

replace：替换字符串特定的字符 返回值（替换后的字符）

charAt：获取字符串中指定索引的字符或根据字符返回索引

split：使用字符串切割另外一个字符串；（返回值是数组）

join：将数组转换为字符串（可用连接符拼接成字符串）
